Selects which output directory will be choosen, according to respect_input_path. If respect_input_path is set to true, the path will be similar as the one into the input directory, plus the filename as directory. If the respect_input_path is set to false, the directory will have the same name as the filename. The output root is out_dir. If the input file is a picture, the behavior will be the same as with respect_input_path set to false.
