Parse the command line to set the variable required by the main process. The variable's default value must have been set before the function is called.

Parameters
argvthe main's argv parameter.
in_paththe input directory or file path buffer variable
out_dirthe output directory path buffer variable
output_name_addonthe name addon buffer variable (default: "_rendered")
timeoutthe timeout variable (default: 0)
blur_filter_sizethe filter size used for blur (see blur_pixel function)
verbosewhether or not text has to be printed
respect_input_pathwhether or not the pictures in the input directory will have a similar path in the output directory.
counry_codethe country code for the plate detection: "eu", "us", "au", "br", "fr", "gb", "in". Default: "eu"
save_plate_infowhether or not the plate's information have to be saved
plate_info_save_paththe path to file or directory to save plate information.
blur_onlywhether or not only bluring should be done. Works only for one file.
blur_only_locationthe location of the areas to be blured. "x11_y11_x12_y12_...x21_y21...". The order is TOP_LEFT, TOP_RIGHT_ BOTTOM_RIGTH, BOTTOM LEFT. Example: 100_150_200_145_250_213_145_200. There must be a multiple of 8 values.
replace_input_filewhether or not you want to replace intput file by the blured image. A copy of the origin file is made with _origin.
